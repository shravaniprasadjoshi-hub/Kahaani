## Overview
**Kahaani** is an interactive platform that brings traditional Indian art forms to life through storytelling.  
Users can explore regional art styles, discover their cultural roots, and experience narratives in a visually immersive way.

---

## Features

### Multi-Language Artist Input
- Supports **9 different languages**
- Artists can express themselves comfortably in their native language

---

### Live Voice Input -> On-Screen Text
- Real-time speech-to-text conversion  
- As the artist speaks, their words appear instantly on screen  
- Preserves **authentic storytelling in native language**

---

### AI-Generated Story Cards
- Converts raw speech into a **compelling narrative**
- Generates short, engaging, and emotionally rich stories
- Makes art more relatable for buyers

---

### AI Image Enhancement
- Artists upload their artwork images  
- System shows **before vs after comparison**
- Automatically improves:
  - Brightness  
  - Clarity  
  - Visual quality  

---

### Cultural Passport for Buyers
- Unique **passport-style UI experience**
- Each purchase adds a **stamp**
- Encourages exploration of diverse art forms

---

### Interactive India Map
- Explore art based on geography  
- Discover regional styles like:
  - Warli  
  - Madhubani  
  - Gond  
  - Toda  
- Visually immersive navigation experience

## Usage
1. Clone the repository and open two terminals
```bash
git clone https://https://github.com/rao-shreyashree/project-kahaani
```
a. Terminal 1:
- navigate into frontend
```bash
cd kahaani/frontend
```
- install dependencies
```bash
npm install
```
- run the frontend
```bash
npm run dev
```
b. Terminal 2: 
- navigate into backend
```bash
cd kahaani/backend
```
- install dependencies
```bash
npm install
```
run the backend
```bash
npm start
```
